Minesweeper
Game

Solo

This project is a modern take on the Minesweeper game, built with HTML, CSS, and JavaScript. Instead of looking like the classic version, it has a cleaner, easier-to-use design. It creates a new minefield every time you play, with different difficulty levels. When you click a square, it shows how many mines are nearby. You can also put flags on squares you think have mines. The game tells you when you win or lose. This shows my skills in making fun, interactive games for the web. It's designed to be easy to understand, with no hints or clues to help you - you have to use your brain! The project is live and the source code is available on my GitHub.
